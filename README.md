# phpstan
PHP Static Analysis Tool - discover bugs in your code without running it!

# Quick start
* Local Build: docker build -t phpstan .
* Local Run: docker run -it phpstan phpstan -V

# Supported tags and respective `Dockerfile` links
* `0.9.2`, `latest`
